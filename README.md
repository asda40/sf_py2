### Структура проекта
Файл Project-2._Ноутбук.ipynb - содержит данные по выполненному проекту
Структура:
1. знакомство с данными;
2. предварительный анализ данных;
3. детальный анализ вакансий;
4. анализ работодателей;
5. предметный анализ.
